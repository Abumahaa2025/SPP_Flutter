# اختبار المحركات داخل التطبيق (beta.18)

بعد تثبيت APK الذي يحمل الختم: `beta-1.0.0-beta.18-engines`

## السيناريو

1. افتح شاشة **الرفع** — تأكد من الختم `beta.18-engines`.
2. ارفع كشوفات الإيجار الحقيقية (Golden).
3. بعد التحليل يظهر **مسار كويل**:
   - Import
   - Property Knowledge
   - Lifecycle
   - Reasoning
   - Executive Report
   - Apply (بانتظار الموافقة)
4. افتح تفاصيل التقرير — النصوص من كويل + بيانات رفعك.
5. اضغط **تحديث المحفظة** — تتحول خطوة Apply إلى مكتملة.
6. تحقق:
   - **المحفظة**: عقار مستورد + وحدات
   - **المستأجرون**: أسماء من الكشوفات
   - **التقارير**: تقرير تنفيذي بنفس `analysis_id` / مختصر كويل

بدون ظهور مسار كويل وأرقام حية = ليست النسخة الجديدة.
