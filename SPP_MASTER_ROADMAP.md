# VISION

SPP is not a traditional property management application.

SPP is an AI-powered Smart Property Operating System.

The system must think before the owner thinks.

The owner should never search for problems.

The Brain discovers problems, explains them, prioritizes them, and recommends the best decision.

The owner only approves or rejects.

---

# PRODUCT PHILOSOPHY

The product must always be:

- Intelligent
- Executive
- Predictive
- Proactive
- Simple
- Fast
- Reliable
- Global

Never build pages just to display information.

Every screen must answer:

"What should I do now?"

# CURRENT DEVELOPMENT RULES

Backend first.

Frontend never contains business logic.

Brain never displays raw data.

Brain explains.

Brain prioritizes.

Brain recommends actions.

Never expose debug messages.

Never expose developer terminology.

Every feature must satisfy at least one of these goals:

Reduce owner workload.

Reduce maintenance cost.

Reduce vacant units.
